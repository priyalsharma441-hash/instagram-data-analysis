# 📊 Instagram Performance Analytics Dashboard

An end-to-end **Data Analytics** project analyzing the Instagram performance of **Sugar Cosmetics** using **Tableau** and **Power BI**. This project transforms social media data into interactive dashboards that help evaluate content performance, audience engagement, and marketing effectiveness.

---

## 🚀 Project Overview

This project focuses on analyzing Instagram metrics to identify trends, measure engagement, and provide actionable insights for improving social media strategy.

---

## 📌 Objectives

* Analyze Instagram performance metrics
* Measure audience engagement
* Compare content performance by post type
* Identify top-performing posts
* Evaluate hashtag effectiveness
* Generate business recommendations

---

## 🛠️ Tools & Technologies

* Tableau
* Power BI
* Microsoft Excel

---

## 📈 Dashboard Features

* KPI Overview
* Monthly Posting Trend
* Post Type Performance
* Day of Week Analysis
* Campaign Type Analysis
* Top 10 Posts by Likes
* Top 10 Posts by Engagement Rate
* Engagement Funnel
* Hashtag Analysis

---

## 💡 Key Insights

* Identified the best-performing content types.
* Compared engagement trends across campaigns.
* Analyzed audience interaction using engagement metrics.
* Provided recommendations to improve reach and engagement.
